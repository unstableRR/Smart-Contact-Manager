This project is made for learning purpose.
So that I can get better clarity on Spring and SpringBoot concepts. It's a basic project but has real-time project features.
MySql database is used in this project and for front-end html, css , js and thymeleaf is used. For back-end Spring
concepts and Spring Boot concepts. Thymeleaf is used for fetching data from front-end dynamically.
This project is for saving contact details in the database and user can delete or update his contacts.
